# BentoHook

A FormIt hook for Bento marketing platform.

## System Settings

| Key                       | Description                |
|---------------------------|----------------------------|
| bentohook.secret-key      | Secret Key from Bento      |
| bentohook.publishable-key | Publishable Key from Bento |
| bentohook.site-uuid       | Site UUID from Bento       |
